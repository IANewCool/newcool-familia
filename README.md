# NewCool Familia - Derecho de Familia Chile

Portal educativo sobre derecho de familia y tribunales en Chile.

## Demo

[https://familia.newcool.io](https://familia.newcool.io)

## Caracteristicas

- Buscador de tribunales de familia
- Calculadora de pension de alimentos
- Guia de divorcio y separacion
- Informacion sobre tuicion
- Mediacion familiar

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
